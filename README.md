# FutAnalise Pro

Análise Inteligente de Futebol

## Objetivo
- Fornecer análise estatística de partidas e times de futebol.
- O projeto é exclusivamente para análise; não oferece recomendações de apostas nem garantias de resultados.
- Foco em visualização de métricas recentes e comparação de equipes.

## Funcionalidades
- Entrada de times e geração de análise simulada.
- Visualização de estatísticas, tendências e recomendações baseadas em dados.
- Interface responsiva com componentes modernos.

## Tecnologias
- Vite, React, TypeScript
- Tailwind CSS, shadcn-ui
- TanStack Query para gerenciamento de dados

## Como executar
```bash
npm install
npm run dev
```
Acesse `http://localhost:8080/`.

## Build de produção
```bash
npm run build
npm run preview
```
Preview em `http://localhost:4173/` (ou porta alternativa).

## Deploy (GitHub Pages)
- URL: `https://renatoredes.github.io/football-insights/`
- Publicar:
```bash
npm run deploy
```
Isso gera o build e publica o conteúdo da pasta `dist` na branch `gh-pages`.

## Aviso
- O conteúdo tem caráter informativo e educacional.
- Não constitui conselho financeiro ou de apostas.
